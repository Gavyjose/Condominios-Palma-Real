# Plan de Implementación - Panel del Propietario Torre 9

Este plan detalla la adición de una vista personalizada para los vecinos, permitiéndoles gestionar su relación con el condominio de forma autónoma.

## Funcionalidades Implementadas

### 1. Vista de Vecino (Owner Panel)
- **Selector de Apartamento**: Menú desplegable para elegir unidad (PB-A a 3-D).
- **Estado de Cuenta**: Deuda anterior, Alícuota ($21.73) y Cuotas de Terraza.
- **Botón de Solvencia**: Habilitado dinámicamente si el saldo es $0.00.

### 2. Notificación de Pagos
- **Formulario**: Fecha, Monto, Referencia y Carga de Comprobantes.
- **Bandeja de Pendientes**: Los pagos notificados aparecen en el panel admin para su aprobación.

### 3. Diseño
- Interfaz 100% responsive para dispositivos móviles.
- Navegación dual integrada en la cabecera.
